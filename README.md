# NST-FRC32: PET Bottle Filament Recycler

*(English version below / Versión en español abajo)*
---

## English

The **NST-FRC32** is an open-source, ESP32-driven machine designed to recycle PET plastic bottles into usable 3D printing filament. 

This repository contains all the necessary files to build, wire, and program the machine.

### Project Status: Work in Progress
This project is currently under active development.
* **Hardware (Mainboard):** Completed (V1.0)
* **Firmware:** In development
* **Mechanical Design (3D Models):** Pending

### Repository Structure
* **`/hardware`**: Contains the schematic, PCB fabrication files (Gerbers, BOM, CPL), and 3D step model for the Nostec Mainboard.
* **`/firmware`**: *(Coming soon)* PlatformIO C++ source code for the ESP32 logic, PID control, and sensor reading.
* **`/mechanical`**: *(Coming soon)* STL and STEP files for the 3D printable parts and structural assembly.

## Español

La **NST-FRC32** es una máquina open-source controlada por un ESP32, diseñada para reciclar botellas de plástico PET y transformarlas en filamento útil para impresión 3D.

Este repositorio contiene todos los archivos necesarios para fabricar, cablear y programar la máquina.

### Estado del Proyecto: En Desarrollo
Este proyecto se encuentra actualmente en fase de desarrollo activo.
* **Hardware (Placa Principal):** Completado (V1.0)
* **Firmware:** En desarrollo
* **Diseño Mecánico (Modelos 3D):** Pendiente

### Estructura del Repositorio
* **`/hardware`**: Contiene el esquemático, archivos de fabricación de la PCB (Gerbers, BOM, CPL) y el modelo 3D STEP de la Mainboard Nostec.
* **`/firmware`**: *(Próximamente)* Código fuente en C++ (PlatformIO) para la lógica del ESP32, control PID y lectura de sensores.
* **`/mechanical`**: *(Próximamente)* Archivos STL y STEP para las piezas imprimibles en 3D y el ensamblaje estructural.
